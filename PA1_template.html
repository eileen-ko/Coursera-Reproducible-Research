Inputting missing values

  1. Calculate the total number of missing values

## [1] 2304
  1. We will use the mean of the 5-minute interval to fill the missing values. First, let’s make a function that will return the mean value for a particular interval.
getMeanStepsPerInterval<-function(interval){
    step_interval[step_interval$intervals==interval,]$steps
}
  1. Now, create a new dataset with the missing values filled with the mean of interval by using for loop. Let’s call the new dataset “datafilled”.
datafilled <- data
for(i in 1:nrow(datafilled)){
    if(is.na(datafilled[i,]$steps)){
        datafilled[i,]$steps <- getMeanStepsPerInterval(datafilled[i,]$interval)
    }
}
